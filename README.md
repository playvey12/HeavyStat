# HeavyStat — Фитнес-трекер

Fullstack приложение для отслеживания тренировок, графиков прогресса и замеров.

**Стек:** React, Node.js, Express, MongoDB, REST API

📋 [Полный план разработки (43 задачи)](ROADMAP.md)

---

## Что умеет

- Создание и кастомизация тренировочных планов
- Графики прогресса по упражнениям
- Трекер тренировок с таймером
- Авторизация и профиль пользователя

---
